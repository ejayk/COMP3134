1. A port number is a way to identify a specific process to which an Internet or other network message is to be forwarded when it arrives at a server.

2. Port numbers range is 0 to 65535, but only port numbers 0 to 1023 are reserved for privileged services and designated as well-known ports. 

3. 16-bit unsigned integer

4. 10 Common Server ports:
	1 - TCPMUX
	7 - ECHO
	18 - MSP (Message Send Protocol)
	20 - FTP DATA
	22 - SSH Remote Login Protocol
	23 - Telnet
	37 - Time
	53 - DNS
	80 - HTTP
	569 - MSN
